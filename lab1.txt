เรื่อง type operator and expression
1. ชื่อตัวแปร (Variable Names) ดารตั้งชื่อตัวแปรควรตั้งให้อ่านแล้วเข้าใจได้ง่ายสอดคล้องกับสิ่งนั้นๆ โดยการตั้งชื่อตัวแปร จะใช้หลักการเดียวกันกับการนำไปใช้ตั้งชื่อฟังก์ชันอีกด้วย
2. ประเภทและข้อมูล (Data Types and Sizes) ประเภทข้อมูลจะใช้แตกต่างกันไปในแต่ละข้อมูลนั้น เช่น ถ้าเป็นตัวอักษรจะใช้ char หรือ ถ้าเป็นตัวเลขจำนวนเต็มจะใช้ int เป็นต้น
ตัวอย่าง Data Types ที่เรามักได้ใช้เช่น char, int, float, double, short int, long int, singed char, unsigned char เป็นต้น หรืแถ้าเราอยากตรวจสอบขนาดตัวแปรและข้อมูลก็สามารถใช้ sizeof ได้
3. การประกาศ (Declarations) ชนิดของตัวแแปรจะต้องถูกกำหนดก่อนถูกนำไปใช้ เช่น ถ้าเราต้องการใช้ตัวแปรที่เป็นจำนวนเต็ม เราก็กำหนดเป็น int ตามด้วยชื่อตัวแปร (int number) เป็นต้น
4. การใช้ printf() หรือ puts() จะมีตัวอ้างอิงไปถึงค่านั้นๆที่เราต้อง output เช่น ถ้าจะให้ output เป็น integer เราจะต้องใช้ %d หรือเราต้องการให้ output เป็น float เราก็ใช้ %f หรือถ้าอยากได้ character เราก็สามารถใช้ %s ได้
5. Character Input and Output เราสามารถใช้ getchar() หรือ putschar()ได้ โดย getchar() จะใช้รับอักขระ 1 ตัวจาก stdin เข้ามาในโปรแกรม และ putchar() ใช้ส่งอักขระ 1 ตัวออกไปยัง stdout
6. Arithmetic Operators คือพวกเครื่องหมายบวก ลบ คูณ หาร (+,-,*,/,%)
7. Relational Operators และ Logical Operators คือเครื่องหมายเปรียบเทียบค่ามากกว่า น้อยกว่า เท่ากับ ไม่เท่ากับ (>, >=, <, <=, ==, !=) และเครื่องหมายแทน and (&&) หรือ แทน or (||) ด้วย
8. Increment and Decrement Operators การเพิ่มค่า เช่น i++ หรือ ++i จะเพิ่มค่าเหมือนกับ i+=1
9. การใช้ Bitwise Operator จะมีพวก and(&&), or(||), exclusive or(^), left shift(<<<) เป็นตัวกระทำ
